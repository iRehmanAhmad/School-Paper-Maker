# Bulk Operations - Quick Reference

## 🚀 Quick Start

### 1. Select Questions
- Click checkboxes next to questions
- Or use "Select All" checkbox in table header
- Bulk actions toolbar appears automatically

### 2. Choose Action
| Button | Action | Shortcut |
|--------|--------|----------|
| ✏️ **Edit** | Update multiple fields | Opens modal |
| 📋 **Duplicate** | Create copies | Instant |
| 💾 **Export** | Download CSV | Instant |
| 🗑️ **Delete** | Remove questions | Confirmation |
| ✖️ **Clear** | Deselect all | Instant |

### 3. Apply Changes
- Review selection count
- Confirm action
- Wait for success message
- ✅ Done!

## 📋 Common Tasks

### Change Difficulty
```
1. Select questions
2. Click "Edit"
3. Check "Difficulty Level"
4. Choose: Easy | Medium | Hard
5. Apply
```

### Move to Chapter
```
1. Select questions
2. Click "Edit"
3. Check "Move to Chapter"
4. Select target chapter
5. Apply
```

### Duplicate & Modify
```
1. Select questions
2. Click "Duplicate"
3. New copies auto-selected
4. Click "Edit" to modify
5. Apply changes
```

### Export Backup
```
1. Select questions (or Select All)
2. Click "Export"
3. CSV downloads automatically
```

### Bulk Delete
```
1. Select questions
2. Click "Delete"
3. Confirm in modal
4. Questions removed
```

## ⚡ Keyboard Shortcuts

| Key | Action |
|-----|--------|
| `Ctrl/Cmd + A` | Select all visible |
| `Escape` | Close modal/Clear selection |
| `Enter` | Confirm action (in modal) |

## 💡 Pro Tips

1. **Use Filters First** - Narrow down before selecting
2. **Export Before Major Changes** - Create backup
3. **Test with Few Items** - Verify before bulk operation
4. **Check Selection Count** - Ensure correct number selected
5. **Use Undo** - If you make a mistake

## 🎯 Field Reference

### Bulk Edit Fields

| Field | Options | Notes |
|-------|---------|-------|
| **Difficulty** | Easy, Medium, Hard | Required for all questions |
| **Bloom's Level** | Remember, Understand, Apply, Analyze, Evaluate | Optional |
| **Question Level** | Exercise, Additional, Past Papers, Examples, Conceptual | Categorization |
| **Chapter** | List of chapters | Moves questions |
| **Topic** | List of topics | Optional assignment |

## ⚠️ Important Notes

- ✅ Only checked fields in Edit modal will be updated
- ✅ Unchecked fields remain unchanged
- ✅ Duplicate adds "(Copy)" to question text
- ✅ Export includes all question data
- ✅ Delete requires confirmation
- ✅ Selection persists across filters
- ✅ Clear selection after successful operation

## 🐛 Quick Troubleshooting

| Problem | Solution |
|---------|----------|
| Toolbar not showing | Select at least one question |
| Edit not applying | Check at least one field |
| Export file empty | Verify questions selected |
| Can't select all | Check filter settings |
| Changes not saving | Check network connection |
